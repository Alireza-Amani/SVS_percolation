Dataset Title: Daily Percolation Volumes from Pan Lysimeters

Principal Investigator: Alexandre R. Cabral
Affiliation: Université de Sherbrooke, Faculty of Engineering, Sherbrooke, Québec, Canada
Email: alexandre.cabral@usherbrooke.ca

Date of Data Collection: 2018-07-01 to 2021-06-30
Geographic Location: Drummondville, Québec, Canada
Data Collection Instruments:
    - pan lysimeters
    - tipping counter (KIPP-100, made by METER Inc., counts 100 mL per tip)

Data Format: .csv
Dataset Name: percolation_daily_20180701_20210630.csv
Dataset Size: 133 KB

Dataset Variables:
    - date: date of the measurement
    - Experimental Plot: plot number, `E1`, `E2`, or `E3`
    - Lysimeter: lysimeter number inside the plot (enclosure), `Lysimeter 1` or `Lysimeter 2`
    - Percolation (mm/day): percolation volume in mm/day

Missing Data: `NaN` values indicate days with missing measurements.

Cite:
Cabral, Alexandre R. (2023). Daily Averaged Soil State Variables and Daily Percolation Volumes from Three Experimental Plots (Soil Covers): 2018-2021 (1.0.0) [Data set]. Zenodo.
